Описание проекта

Forest-Vision v2 — это глубокая нейронная сеть на базе архитектуры Vision Transformer (ViT), специально обученная для сегментации лесных массивов по спутниковым снимкам. Модель способна различать более 15 видов деревьев и определять уровень влажности почвы по цветовым спектрам.

Идеально подходит для экологического мониторинга и предотвращения лесных пожаров на ранних стадиях.

Пример использования

import aibloom
from aibloom.models import ForestVision

# Загрузка модели из нашего сада
model = ForestVision.from_pretrained("forest-vision-v2")

# Анализ снимка
image = "satellite_forest_01.jpg"
prediction = model.predict(image)

print(f"Обнаружено: {prediction.labels}")
